# 777Keybord
> 一个基于STM32G431CBU6的7键迷你磁轴机械键盘

*音游痴专用(*  

***支持的功能***  
  键盘支持OLED屏幕,UI使用[bdth-7777777](https://github.com/bdth-7777777)大佬的框架  
  使用芯片内置ADC读取霍尔传感器模拟量  
  支持2种rt触发(目前还在测试)  
  支持通过OLED菜单修改键程  
  外挂FLASH记忆键盘设置  
  使用旋转编码器操作菜单进行设置
